# Calculator Application
---
link: [click here to visit](https://calculator-nine-orpin-70.vercel.app/)

A simple, responsive, and visually clean calculator built with HTML, CSS, and JavaScript. It performs basic arithmetic operations, percentage calculations, and includes an intuitive user interface.

## Features

- Basic operations: addition, subtraction, multiplication, division
- Percentage calculation with contextual logic (e.g., 50 + 10% = 55)
- Clear entry (AC) and backspace (←) functions
- Decimal point and double-zero (00) input
- Error handling for invalid expressions
- Responsive design that adapts to smaller screens

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, media queries)
- Vanilla JavaScript

## How to Use

1. Clone the repository or download the source code.
2. Open the `index.html` file in any modern web browser.
3. Use the on-screen buttons to enter numbers and operations.
4. Press `=` to see the result.
5. Use `AC` to reset the display or `←` to remove the last character.
